Problem

낚시 시장은 시즌성·대회 문화·고관여 장비·세대별 사용자 정착이 모두 작용하는 독특한 도메인. 단순 커머스나 단순 커뮤니티로는 임팩트가 안 났습니다.

Hypothesis

"슈퍼앱"으로 예약·쇼핑·커뮤니티·중고·머니·챔피언십·홈·마이·알림을 통합할 때만 사용자 시간 점유율과 결제 락인을 동시에 잡을 수 있습니다.

Solution

  • 9개 도메인 역할 분리 — 매출 축(쇼핑 + 예약 = 95%) + 락인 축(커뮤니티·중고·머니 = DAU/체류) + 전환 축(홈·검색·마이·알림)
  • 매출 채널 4분산 — 스마트스토어 73% + 자사몰 20% + 예약·대회·외부 7%
  • 머니 적립률 도메인별 차등화(예약 1.5% / 쇼핑 1% / 커뮤니티 0.5%) — 마케팅 ROI · 운영 어뷰징 데이터 합의 후 결정
  • 임원진 / 운영팀 / 외부 파트너(TMAP)와의 우선순위 정렬을 CPO가 직접

Result

  • 2025 연매출 14.26억 · 사업 KPI 97.6% 달성
  • 월 KPI 달성률 4월 119% / 8월 124% / 9월 113%
  • 자사몰 ARPPU 67,020 → 89,925원 (+34%)
